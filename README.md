# 📊 Portfólio | Thiago Oliveira Maia

**Analista de Dados & Business Intelligence**

Portfólio profissional desenvolvido para apresentar meus projetos de Business Intelligence, Engenharia de Dados e Desenvolvimento Front-End. O objetivo desta página é demonstrar como aplico a ótica da Engenharia de Software e conceitos de UI/UX Analítico na construção de Produtos de Dados.

🌐 **Acesse online:** [maia-th.vercel.app](https://maia-th.vercel.app) 


## 🚀 Tecnologias Utilizadas

Este projeto foi construído sem o uso de frameworks pesados, focando em performance, acessibilidade e carregamento rápido:

* **HTML5:** Estrutura semântica e otimizada para SEO.
* **CSS3:** Estilização com variáveis (Custom Properties) e Flexbox/Grid.
* **JavaScript (Vanilla):** Lógica de renderização de dados, controle de temas e interatividade de mídia.
* **Phosphor Icons:** Biblioteca de ícones moderna e minimalista.
